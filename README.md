# Interactive Quiz Web Application

This is an interactive and user-friendly Quiz Web Application developed using HTML, CSS, and JavaScript.  
The application displays questions, collects user answers, and shows the final score at the end of the quiz.

---

## Features

- Displays quiz questions one by one
- Supports multiple types of questions:
  - Single Select (Radio buttons)
  - Multi Select (Checkboxes)
  - Fill in the Blanks (Text input)
- Collects user responses
- Automatically calculates the score
- Displays the final score after quiz completion
- Simple and clean user interface

---

## Technologies Used

- HTML – For structuring the web page
- CSS – For styling and layout
- JavaScript – For quiz logic, scoring, and interactivity
